# clojure-editor

Subagent for editing Clojure files using Backseat Driver structural editing tools.

Version: 0.1.1

## Agents

### Clojure-editor

A subagent for editing Clojure files effectively. Use when using Backset Driver editing tools. Use when editing Clojure files, regardless of dialect or runtime. This agent takes an edit plan and carries it out, using a structured process to validate the plan, apply edits, check for problems, and report on the results.
